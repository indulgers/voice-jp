# voice-jp 安装与使用指南

> 给完全不熟悉编程的朋友的安装手册。预计总耗时 **15–25 分钟**（含模型自动下载）。

## 你需要准备的东西

- 一台 Mac（Apple Silicon，M1/M2/M3/M4 均可）
- 大约 4GB 可用硬盘空间（模型 3GB + 缓存）
- 微信账号（可以扫码登录）

---

## 步骤 1：装 WeFlow（微信桥）

voice-jp 自己不会连微信。它需要 WeFlow 帮忙读取微信里的语音消息。

1. 打开 https://github.com/hicccc77/WeFlow/releases
2. 下载最新版的 `WeFlow-x.x.x-arm64.dmg`
3. 双击 dmg，把 **WeFlow.app** 拖进 **Applications** 文件夹
4. 打开 Applications，**右键点击 WeFlow → 「打开」**（双击会被系统拦截，必须右键）
5. 弹出"无法验证开发者"的对话框，点 **「打开」**
6. 在 WeFlow 里**扫码登录微信**（用手机微信扫一下）
7. 登录成功后，打开 WeFlow 的 **设置**：
   - 勾选 **「HTTP API」** （开启接口服务）
   - 勾选 **「Message Push」** （开启消息推送）
   - 复制旁边显示的 **access_token**（一长串字母数字，例如 `0123456789abcdef0123456789abcdef`）—— 等下要粘贴

保持 WeFlow 开着，**不要关**。

---

## 步骤 2：装 voice-jp

1. 把 **voice-jp.dmg**（22MB，从我这里发给你的文件）保存到 Downloads
2. 双击 dmg，把 **voice-jp.app** 拖进 **Applications**
3. 打开 Applications，**右键点击 voice-jp → 「打开」**
4. 弹出"无法验证开发者"的对话框，点 **「打开」**
5. 几秒后，**浏览器会自动弹出一个标签页**（地址是 `http://localhost:7788`）

如果浏览器没自动弹，自己手动复制粘贴这个地址打开。

---

## 步骤 3：跟随浏览器向导（4 步）

浏览器里会出现一个引导页：

### 第 1 步 — 「WeFlow 起動」
应该自动显示 ✓（绿色）。如果显示红色"接続できません"，说明 WeFlow 没在跑，回去步骤 1 检查。

### 第 2 步 — 「access_token 貼り付け」
粘贴你在 WeFlow 设置里复制的那串 token，点保存。

### 第 3 步 — 「モデル ダウンロード」
点「ダウンロード開始」。会下载约 3GB 的语音识别模型到本地。**5–15 分钟**，看你网速。下载条会实时更新进度，不用守着。

### 第 4 步 — 「監視する連絡先」
- **語言** 选「日本語」（如果你要转中文就选「中国語」）
- 在搜索框输入你日本朋友的微信名 / 备注 / 微信号，从列表里点选 —— 可以选多个人
- 点「保存して完了」

完事了！主界面会出现。

---

## 使用

只要 WeFlow 和 voice-jp 都开着，**白名单内联系人**发的每条语音消息都会自动出现在 voice-jp 的浏览器界面里，几秒后显示转写文字。可以播放原音、复制文字。

我推荐：把 voice-jp 的浏览器窗口拖窄、放在屏幕角落、按 Cmd+Shift+T 之类的快捷键固定（你的浏览器支持的话）。

## 调整设置

想加/减联系人，或改语言：在 voice-jp 的浏览器界面里点 **「⚙ 設定」** 展开，重新选完点保存。

## 关闭和重启

- **关 voice-jp**：在 Dock 上右键 voice-jp → 退出
- **关 WeFlow**：同理
- 下次想用，分别打开就行。**不会自动开机启动**（这是有意的，避免后台一直跑）

## 数据保存在哪

聊天转写结果保存在你 Mac 上的：`~/Library/Application Support/voice-jp/`

包含：
- `voice-jp.db` — 转写文本数据库
- `cache/` — 下载的语音文件
- `models/ggml-large-v3.bin` — 3GB 模型文件

要彻底清除就把这个文件夹删了。

## 常出问题

| 现象 | 检查 |
|---|---|
| 浏览器显示"WeFlow に接続できません" | WeFlow 没开 / 退出了 / 没登录 → 重启 WeFlow，重新登录 |
| 朋友发了语音但没出现 | 1) 微信 PC 端要登录并接收到这条消息 2) 这个联系人不在白名单里 → 设置里加上 |
| voice-jp 打开报"无法验证开发者" | 一定要**右键 → 打开**，不能双击。第一次确认后下次双击就行了 |
| 模型下载失败 | 重新点「ダウンロード開始」会自动续传 |
| 想转其他语言（不是日语） | 设置里改「言語」下拉，选 ja / zh / auto |

遇到任何其他问题，截图发我。
